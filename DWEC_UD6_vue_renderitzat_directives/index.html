    Renderitzat condicional

    Clar, entenc que prefereixes que la resposta sigui en català. Aquí tens l'explicació del renderitzat condicional en Vue 3:

    El renderitzat condicional en Vue 3 es basa en mostrar o amagar elements o blocs de codi basant-se en condicions. Pots utilitzar diverses directrius i tècniques per aconseguir el renderitzat condicional en Vue 3. Aquí tens algunes formes comunes de fer-ho:

    1. Directiva v-if i v-else

    La directiva v-if s'utilitza per renderitzar un element o bloc de codi si una expressió és certa. Pots utilitzar v-else per renderitzar un altre element o bloc de codi si l'expressió és falsa.

    <template>
  <div>
    <p v-if="mostrarMissatge">Hola, aquest missatge es mostra si mostrarMissatge és cert.</p>
    <p v-else>No es mostrarà si mostrarMissatge és fals.</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      mostrarMissatge: true,
    };
  },
};
</script>

    2. Directiva v-show

    La directiva v-show també s'utilitza per mostrar o amagar elements, però a diferència de v-if, els elements amb v-show sempre estan presents al DOM, simplement s'oculten o mostren mitjançant CSS.

    <template>
  <ul>
    <li v-for="item in llistaDItems" :key="item.id">
      <span v-if="item.actiu">{{ item.nom }}</span>
      <span v-else style="text-decoration: line-through;">{{ item.nom }}</span>
    </li>
  </ul>
</template>

<script>
export default {
  data() {
    return {
      llistaDItems: [
        { id: 1, nom: 'Element 1', actiu: true },
        { id: 2, nom: 'Element 2', actiu: false },
        // ...
      ],
    };
  },
};
</script>

    Aquestes són només algunes de les formes en què pots fer el renderitzat condicional en Vue 3. La elecció entre elles dependrà de la situació i del comportament específic que vulguis aconseguir a la teva aplicació.

    Renderitzar bucles amb v-for

    En Vue 3, pots renderitzar bucles amb la directiva v-for. Aquesta directiva t'ajuda a iterar sobre una llista o objecte i renderitzar dinàmicament elements a la plantilla. Aquí tens un exemple senzill que demostra com utilitzar v-for per renderitzar una llista d'elements:

    <template>
  <div>
    <h2>Llista d'Elements</h2>
    <ul>
      <li v-for="(element, index) in llistaDElements" :key="index">
        {{ index + 1 }}. {{ element.nom }}
      </li>
    </ul>
  </div>
</template>

<script>
export default {
  data() {
    return {
      llistaDElements: [
        { id: 1, nom: 'Element 1' },
        { id: 2, nom: 'Element 2' },
        { id: 3, nom: 'Element 3' },
      ],
    };
  },
};
</script>

    En aquest exemple:

    • La directiva v-for s'aplica a l'element li per iterar sobre la llista d'elements llistaDElements.
    • La sintaxi (element, index) in llistaDElements indica que es farà un bucle sobre cada element de la llista i també es proporcionarà l'índex d'aquest element.
    • Amb :key="index", s'estableix una clau única per cada element de la llista. Això ajuda a Vue a identificar els elements de manera eficient durant les actualitzacions.

    <template>

    <h2>Detalls del Producte</h2>
    <ul>
      <li v-for="(valor, clau) in detallsProducte" :key="clau">
        {{ clau }}: {{ valor }}
      </li>
    </ul>
  </div>

<script>
export default {
  data() {
    return {
      detallsProducte: {
        marca: 'Marca X',
        model: 'Model Y',
        preu: '$100',
      },
    };
  },
};
</script>

    En aquest cas, v-for itera sobre les propietats de l'objecte detallsProducte i mostra el nom de la propietat (clau) i el seu valor (valor).

    Recorda que és important utilitzar una clau única amb v-for per garantir un renderitzat eficient i correcte dels elements. La clau és especialment important quan estàs iterant sobre una llista d'elements que pot canviar dinàmicament.
