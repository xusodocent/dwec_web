    El router de Vue

    En Vue 3, el router fa referència al router oficial de Vue, anomenat Vue Router. Vue Router és una llibreria que s'utilitza per a la navegació en aplicacions Vue d'una sola pàgina (SPA). Facilita la gestió de les transicions entre diferents "pàgines" o components en una aplicació Vue, permetent que els usuaris naveguin entre diferents vistes sense haver de recarregar la pàgina.

    Algunes característiques clau de Vue Router inclouen:

    1. Enrutament Declaratiu: Vue Router permet definir rutes de forma declarativa en lloc de imperativa. Pots emparellar rutes amb components Vue específics, cosa que facilita l'associació de components amb rutes específiques.

    2. Navegació Basada en Components: Cada ruta pot tenir un component associat, i quan l'usuari navega a aquesta ruta, el component es renderitza a l'àrea de la interfície d'usuari designada per mostrar contingut dinàmic.

    3. Navegació Programàtica: Pots canviar de ruta programàticament mitjançant l'ús de mètodes proporcionats per Vue Router. Per exemple, pots utilitzar router.push() per navegar a una nova ruta.

    4. Paràmetres de Ruta: Vue Router permet definir paràmetres dinàmics en les rutes. Aquests paràmetres poden ser capturats i utilitzats en el component associat a aquesta ruta.

    5. Anidament de Rutes: Pots anidar rutes per crear una jerarquia de vistes a la teva aplicació. Això és útil quan necessites dividir la teva aplicació en mòduls o seccions.

    6. Navegació amb Historial: Vue Router gestiona la navegació amb historial, la qual cosa significa que pots utilitzar la barra de navegació del navegador per retrocedir i avançar a la pila d'historial.

    // main.js

import { createApp } from 'vue'
import App from './App.vue'
import { createRouter, createWebHistory } from 'vue-router'

import Home from './components/Home.vue'
import About from './components/About.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

const app = createApp(App)
app.use(router)
app.mount('#app')

    En aquest exemple:

    • S'importa createRouter i createWebHistory de vue-router.
    • Es defineixen rutes associant rutes amb components específics.
    • Es crea una instància del router amb createRouter.
    • La instància del router s'utilitza a l'aplicació mitjançant app.use(router).

    Amb aquesta configuració, pots començar a utilitzar el router a la teva aplicació Vue 3 per gestionar la navegació entre diferents vistes. Per obtenir més informació i detalls, consulta la documentació oficial de Vue Router.

    Instal·lació

    Per instal·lar Vue Router 4 (la versió compatible amb Vue 3), pots utilitzar npm o yarn. Aquí tens com fer-ho amb npm:

    1. Crea un projecte Vue 3: Assegura't que ja tens un projecte Vue 3 creat. Si no, pots utilitzar la següent comanda per crear un projecte amb vue-cli:
    Bash
    1
    vue create nom_del_projecte

    Segueix les instruccions per configurar el projecte.

    1. Instal·la Vue Router: Accedeix al directori del teu projecte i executa la següent comanda per instal·lar Vue Router:
    Bash
    1
    npm install vue-router@4

    Si utilitzes yarn, pots utilitzar aquesta comanda:

    Bash
    1
    yarn add vue-router@4
    1. Configura Vue Router en el teu projecte: Després d'instal·lar Vue Router, has de configurar-lo en el teu projecte Vue. Obre l'arxiu src/main.js (o el teu arxiu d'entrada principal) i afegeix el següent codi:
    JavaScript

    import { createRouter, createWebHistory } from 'vue-router'
import HomeView from '../views/HomeView.vue'

 const routes = [
 {
     path: '/',
     name: 'home',
     component: HomeView
 },
 {
     path: '/about',
     name: 'about',
     // route level code-splitting
     // this generates a separate chunk (about.[hash].js) for this route
     // which is lazy-loaded when the route is visited.
     component: () => import(/* webpackChunkName: "about" */ '../views/AboutView.vue')
 }
 ]

 const router = createRouter({
 history: createWebHistory(process.env.BASE_URL),
 routes
 })

 export default router

    On Home i About són components que has de crear i importar.

    Ara ja has instal·lat i configurat Vue Router amb èxit al teu projecte Vue 3. Pots començar a utilitzar Vue Router per gestionar les rutes de la teva aplicació. Consulta la documentació oficial de Vue Router per obtenir més informació sobre com utilitzar-lo.

    // main.js
import { createApp } from 'vue';
import App from './App.vue';
import ItemDetails from './components/ItemDetails.vue';
import { createRouter, createWebHistory } from 'vue-router';

const router = createRouter({
  history: createWebHistory(),
  routes: [
    { path: '/', redirect: '/item/1' }, // Redirigeix a la primera pàgina amb ID 1
    { path: '/item/:id', component: ItemDetails, props: true },
  ],
});

const app = createApp(App);
app.use(router);
app.mount('#app');

    En aquest exemple, la ruta '/item/:id' indicarà que el component ItemDetails es renderitzarà quan es visiti una URL que segueixi aquest patró. El paràmetre id estarà disponible al component a través de $route.params.id. Pots adaptar aquest exemple segons les teves necessitats específiques.
