    1. Validació

    Cal tindre en compte que Javascript nasqué amb l'objectiu principal de validar formularis. Amb l'aparició d'AJAX han augmentat molt més les prestacions del llenguatge, tot i això la validació de formularis seguix sent un dels principals usos.

    evitar reenviament

    Es pot evitar enviar un formulari dues vegades per culpa de connexions lentes, doble polsació, etc. Una forma és amb un botó button, deshabilitant-lo abans d'enviar i canviant el missatge del botó.

    HTML
    1
2
3
4
    <form id="formulario" action="#">
  ...
  <input type="button" value="Enviar" onclick="this.disabled=true; this.value=’Enviando...’; this.form.submit()" />
</form>

function validarFormulario(evento) {
  evento.preventDefault();
  var usuario = document.getElementById('usuario').value;
  if(usuario.length == 0) {
    alert('No has escrito nada en el usuario');
    return;
  }
  var clave = document.getElementById('clave').value;
  if (clave.length < 6) {
    alert('La clave no es válida');
    return;
  }
  this.submit();
}
