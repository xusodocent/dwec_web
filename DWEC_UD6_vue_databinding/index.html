    Data binding

    Que és data binding?

    Data binding

    En Vue.js 3, el "data binding" (vinculació de dades) es refereix a la capacitat de mantenir sincronitzats els dades entre el model (la capa de dades) i la vista (la capa de presentació). Vue.js facilita aquest procés mitjançant un sistema de reactivitat que permet actualitzar automàticament la interfície d'usuari quan les dades subjacents canvien, i viceversa.

    En el context de Vue.js 3, hi ha dos tipus principals de data binding:

    One-Way Binding (Vinculació Unidireccional):

    • Interpolació (Interpolation): Pots usar la sintaxi de dobles claus {{ }} per realitzar la vinculació unidireccional de dades des del model a la vista. Per exemple:

      HTML
      1
      <p>{{ message }}</p>

      On message és una propietat en el model de dades.

    Two-Way Binding (Vinculació Bidireccional):

    • v-model: Aquesta directiva s'utilitza per aconseguir la vinculació bidireccional entre un input i una propietat en el model. Per exemple:

      HTML
      1
      <input v-model="message">

      En aquest cas, qualsevol canvi a l'input actualitzarà la propietat message en el model, i qualsevol canvi en message en el model també es reflectirà a l'input.

    Vue.js utilitza un sistema de reactivitat basat en l'objecte Proxy per detectar canvis en les dades i actualitzar automàticament les vistes afectades. Això significa que no necessites preocupar-te de manipular manualment el DOM; Vue ho fa per tu.

    <template>
  <div>
    <h2>Exemple de Data Binding en una Taula amb Vue 3</h2>

    <!-- Taula utilitzant v-for per iterar sobre l'array de dades -->
    <table>
      <thead>
        <tr>
          <th>ID</th>
          <th>Nom</th>
          <th>Edat</th>
        </tr>
      </thead>
      <tbody>
        <tr v-for="persona in personas" :key="persona.id">
          <td>{{ persona.id }}</td>
          <td>{{ persona.nom }}</td>
          <td>{{ persona.edat }}</td>
        </tr>
      </tbody>
    </table>
  </div>
</template>

<script>
export default {
  data() {
    return {
      personas: [
        { id: 1, nom: 'Joan', edat: 25 },
        { id: 2, nom: 'Anna', edat: 30 },
        { id: 3, nom: 'Pere', edat: 28 },
      ],
    };
  },
};
</script>

<style scoped>
/* Estils específics per a aquest component, si és necessari */
</style>

    En aquest exemple:

    • La propietat personas en data conté un array d'objectes, cadascun representant una persona amb id, nom, i edat.

    • S'utilitza la directiva v-for per iterar sobre l'array de personas i crear dinàmicament les files de la taula.

    • :key="persona.id" s'utilitza per proporcionar una clau única a cada fila, el que ajuda a Vue a fer un seguiment eficient dels elements durant les actualitzacions.

    Cada cop que canvies les dades a l'array personas, la taula es actualitzarà automàticament gràcies al sistema de reactivitat de Vue. Aquest és un exemple bàsic, i pots expandir-lo segons les necessitats específiques de la teva aplicació.

    <template>
  <div>
    <h2>Exemple de v-bind en Vue 3</h2>

    <!-- Utilitzant v-bind per enllaçar dinàmicament l'atribut 'href' -->
    <a :href="url" target="_blank">Enllaç Extern</a>

    <!-- Utilitzant v-bind per enllaçar dinàmicament l'estil 'color' -->
    <p :style="{ color: textColor }">Aquest text té un color dinàmic.</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      url: 'https://www.example.com',
      textColor: 'blue',
    };
  },
};
</script>

<style scoped>
/* Estils específics per a aquest component, si és necessari */
</style>

    En aquest exemple:

    • Utilitzem :href="url" per enllaçar dinàmicament l'atribut href d'un element <a>. Això fa que l'enllaç apunti a la URL continguda a la variable url.

    • Utilitzem :style="{ color: textColor }" per enllaçar dinàmicament l'estil color d'un element <p>. Això fa que el color del text sigui dinàmic, canviant segons el valor contingut a la variable textColor.

    Aquests són exemples senzills, però v-bind pot ser utilitzat per enllaçar dinàmicament gairebé qualsevol atribut o propietat d'un element HTML o component de Vue. Pots fer servir expressions, variables, o fins i tot objectes o arrays per definir el valor que es vincula.

    v-bind: i :

    En Vue 3, v-bind: és equivalent a utilitzar el modificador curt :. Ambdues formes són utilitzades per aplicar la directiva v-bind.

    Per exemple, aquestes dues declaracions són equivalents:

    HTML
    1
2
3
4
5
    <!-- Utilitzant v-bind: -->
<a v-bind:href="url">Enllaç Extern</a>

<!-- Utilitzant : (modificador cort) -->
<a :href="url">Enllaç Extern</a>

    Ambdues maneres produeixen el mateix resultat i són comunes en la comunitat de desenvolupadors de Vue. La versió amb : (modificador cort) és més concisa i és molt utilitzada en la pràctica. Pots triar la forma que trobis més llegible o amb la qual et sentis més còmode.

    <template>
  <div>
    <h2>Exemple de v-model en Vue 3 amb un Input</h2>

    <!-- Utilitzant v-model amb un input -->
    <label for="inputText">Introdueix un text:</label>
    <input v-model="text" id="inputText" type="text" />
    <p>Text introduït: {{ text }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      text: '',
    };
  },
};
</script>

<style scoped>
/* Estils específics per a aquest component, si és necessari */
</style>

    En aquest exemple:

    • Utilitzem v-model="text" a l'input per establir una vinculació bidireccional amb la variable text del data del component. Això significa que qualsevol canvi a l'input actualitzarà automàticament la variable text i viceversa.

    • Mostrem el valor actual de text a la pàgina amb la interpolació {{ text }}.

    Aquesta és una forma senzilla d'utilitzar v-model amb un input, i aquesta directiva també es pot utilitzar amb altres tipus d'elements com ara checkboxes, radios i selects. La directiva v-model proporciona una manera concisa i convenient de gestionar l'estat dels formularis en les aplicacions Vue.
