    UD5 AJAX en Promeses

    1. Que és una promesa?

    Vegem com ho expliquen a lenguajeJS.com;

    Les promeses són un concepte per resoldre el problema d'assincronia de manera molt més elegant i pràctica que, per exemple, utilitzant funcions callbacks directament.

    Com indica el seu propi nom, una promesa és una cosa que, en principi pensem que es complirà, però en el futur poden passar diverses coses:

    ;

    • La promesa es compleix (promesa resolta)
    • La promesa no es compleix (promesa es rebutja)
    • La promesa es queda en un estat incert indefinidament (promesa pendent)

    const promesa = new Promise((resolve, reject) => {
  // Simulem una operació asíncrona
  setTimeout(() => {
    const exit = true; // Suposem que l'operació va bé
    if (exit) {
      resolve('Dades de la promesa'); // La promesa es compleix
    } else {
      reject('Hi ha hagut un error'); // La promesa es rebutja amb un missatge d'error
    }
  }, 2000); // Esperem 2 segons abans de resoldre o rebutjar la promesa
});

promesa.then((resultat) => {
  console.log('Promesa completada:', resultat);
}).catch((error) => {
  console.error('Promesa rebutjada:', error);
});

    En aquest exemple, creem una promesa que simula una operació asíncrona mitjançant un temporitzador (setTimeout). Si l'operació té èxit, la promesa es resol amb les dades. Si hi ha un error, la promesa es rebutja amb un missatge d'error. A continuació, utilitzem els mètodes .then() i .catch() per gestionar la resolució o el rebuig de la promesa.

    function carregaQuadrats(url){
            promesa = new Promise((resolve,reject)=>{
                xhr=new XMLHttpRequest();
                xhr.onreadystatechange=()=>{
                    if (xhr.readyState == 4){
                        if (xhr.status == 200){
                            //resposta=JSON.parse(xhr.responseText);
                            //console.log(resposta[0].url);
                            //console.log(xhr.responseText);
                            resolve(xhr.responseText);
                        }else {reject();}
                    }
                }
                xhr.open("GET", url, true);
                xhr.send();
            });
            return promesa;
        }

        function exitPromesa(resp){
            //console.log(resp);
            vector = JSON.parse(resp);
            //alert(vector.length);
            cadena="";
            vector.forEach(elem=>{
                cadena += "<div><p>"+elem.title+"</p></div>";
            });
            document.getElementById("contenedor").innerHTML=cadena;
        }

        function mostraError(){
            alert("Error en la petició...");
        }
