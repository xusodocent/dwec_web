<script>
import TaulaComponent from './components/TaulaComponent.vue';

export default {
  components: {
    TaulaComponent,
  },
  data() {
    return {
      dadesPersones: [
        { id: 1, nom: 'Joan', edat: 25 },
        { id: 2, nom: 'Anna', edat: 30 },
        { id: 3, nom: 'Pere', edat: 28 },
      ],
    };
  },
};
</script>

<style>
/* Estils globals de l'aplicació, si és necessari */
</style>

    En aquest exemple:

    • S'ha afegit una propietat persones al component TaulaComponent utilitzant la clau props. Aquesta propietat serà utilitzada per mostrar les dades a la taula.

    • A l'aplicació principal, s'ha creat una propietat dadesPersones a data i s'ha passat aquesta propietat com a propietat al component fill utilitzant la sintaxi :persones="dadesPersones".

    Ara, el component de taula rep les dades com a propietat i es pot utilitzar d'aquesta manera per tal de passar dades entre components a Vue.

    Pas de dades d'un component fill a un component pare

    A Vue.js, per passar dades d'un component fill a un component pare, normalment es fa utilitzant esdeveniments (events). El component fill emet un esdeveniment, i el component pare escolta aquest esdeveniment per capturar les dades que es transmeten.

    A continuació, es mostra un exemple senzill d'aquesta dinàmica:

    <template>
  <div>
    <button @click="enviarDades">Enviar Dades</button>
  </div>
</template>

<script>
export default {
  methods: {
    enviarDades() {
      // Simulem alguna acció o dades que es volen transmetre
      const dades = {
        missatge: 'Hola desde el fill',
      };

      // Emitim un esdeveniment anomenat 'dades-enviades' amb les dades com a paràmetre
      this.$emit('dades-enviades', dades);
    }
  }
};
</script>

    <template>
  <div>
    <h2>Component Pare</h2>

    <!-- Afegim el component fill i escoltem l'esdeveniment 'dades-enviades' -->
    <FillComponent @dades-enviades="capturarDades" />

    <p>Dades rebudes: {{ dadesRebudes?.missatge }}</p>
  </div>
</template>

<script>
import FillComponent from './FillComponent.vue';

export default {
  components: {
    FillComponent,
  },
  data() {
    return {
      dadesRebudes: null,
    };
  },
  methods: {
    // Mètode que s'executarà quan s'escolti l'esdeveniment 'dades-enviades'
    capturarDades(dades) {
      // Dades rebudes del component fill
      this.dadesRebudes = dades;
    }
  }
};
</script>

    En aquest exemple:

    • En el component fill (FillComponent), quan es fa clic al botó, s'emeteix l'esdeveniment dades-enviades amb les dades desitjades.

    • En el component pare (PareComponent), s'afegeix el component fill i es posa un @dades-enviades="capturarDades" per escoltar l'esdeveniment. Quan es produeix l'esdeveniment, s'executa el mètode capturarDades que captura les dades i les guarda a la propietat dadesRebudes, que pot ser utilitzada a la plantilla.

    Aquest és un patró comú per passar dades dels components fills als components pares en Vue.js.
