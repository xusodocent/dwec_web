    Router

    Fake API (0,5p)

    Munta una Fake API amb aquest JSON per a poder fer de servidor.

    Menú (1p)

    Caldrà tindre un menú sempre visible entre les diferents rutes. Haurà de ser un component

    Vista pricipal (1p)

    La vista per defecte de la vostra aplicació serà una vista amb el vostre nom, cognom i dni.

    Llista d'empreses (2,5p)

    Has de mostrar les dades bàsiques de les empreses (nom, ciutat i e-mail) en cards.

    Ordre alfabètic

    Les empreses les ha de mostrar per ordre alfabètic

    Components

    Les cards hauran de ser components anomentats targetes

    Detall empresa (1p)

    Cal fer una vista que mostre totes les dades de l'empresa.

    Nova empresa (2p)

    S'ha de validar el formulari abans d'enviar per a que tots els camps de text siguen correctes (tinguen una longitud mínima de 3).

    Botó deshabilitat

    Deshabilitar el botó d'enviar fins que el formulari siga correcte.

    Es motrarà un avís mitjançant dos <div> que seran visibles o no si s'ha creat correctament l'empresa en la base de dades.

    Modificar empresa (0,5p)

    El seu comportament ha de ser igual que el de crear empresa.

    Esborrar (0,5p)

    Cal implementar la funcionalitat d'esborrar les empreses. Ha de mostrar també els avisos amb <div> que seran visibles o no en funció de si s'ha realitzat l'operació correctament.

    Login (1p)

    Simula un sistema de login i protegeix alguna de les vistes de l'aplicació.
